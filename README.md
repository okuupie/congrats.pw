# congrats.pw

**congrats.pw - Catlavan Client Fixer**

Simple tool that will help fix Catlavan Client (Minecraft)

## Main Tab

*	**Repair** will help you with any error

## Extra Tab

*   **-temp** will clear your temp folder (%LOCALAPPDATA%\Temp\* or %TEMP%\*)
*	**-rewrite** will rewrite configuration file of Catlavan Client
*	**-drivers** will download runtimes to your pc
*	**-zapret** will stop zapret and WinDivert services
*	**-killprogs** will close all programs in list (firefox.exe, browser.exe (Yandex), chrome.exe, discord.exe, etc.)

## Extra Tab \ CMD Utils

*	**sysinfo** will open cmd with command: "systeminfo"
*	**ping** pings the site "catlavan.net" (Catlavan Client)

## Info Tab

* Contains dev. contacts (discord)

## Info Tab \ PC Tab

*	**OS:** Windows 10/11 (build version)
*	**RAM:** memory in MB

## Custom Tab

** There you can customise GUI with a hex color
** And also you can rename tabs.

## Privacy

Program collects **anonymous** system info (OS, RAM) to improve compatibility.  
You can disable it by running with `-noinfo` or `-ni` flag:  
cmd -> `congrats.pw.exe -noinfo` or `congrats.pw.exe -ni`
